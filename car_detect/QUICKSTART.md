# 快速开始指南

## 🎯 5分钟快速体验

### Windows用户

1. **解压项目文件**到任意目录
2. **双击运行** `start.bat`
3. **打开浏览器**访问 http://127.0.0.1:8000
4. **上传车辆图片**开始检测

### Linux/Mac用户

```bash
# 1. 进入项目目录
cd car_detect

# 2. 赋予执行权限
chmod +x deploy.sh

# 3. 运行部署脚本（需要sudo权限）
sudo ./deploy.sh

# 4. 访问系统
# 浏览器打开: http://YOUR_SERVER_IP:8000
```

## 📝 首次使用

1. **选择车辆类型**：轿车/SUV/卡车等
2. **上传图片**：点击或拖拽车辆照片
3. **等待分析**：AI自动识别车辆状况
4. **查看报告**：获取专业检测报告

## 💡 使用技巧

1. **图片质量**：使用清晰、光线充足的图片效果更好
2. **拍摄角度**：建议从正面、侧面、背面多角度拍摄
3. **文件大小**：单张图片不超过10MB
4. **支持格式**：JPG、PNG、GIF、BMP

## ❓ 常见问题

**Q: 为什么检测结果都是模拟数据？**  
A: 当前版本为演示版本，YOLO模型和LLM集成功能已预留接口，可根据需求接入真实模型。

**Q: 如何修改服务器端口？**  
A: 编辑 `.env` 文件，修改 `PORT=8000` 为其他端口号。

**Q: 上传文件保存在哪里？**  
A: 文件保存在 `static/upload/` 目录下，文件名已做唯一化处理。

**Q: 如何查看日志？**  
A: 日志文件在 `logs/` 目录，按天自动切割。

**Q: 生产环境如何部署？**  
A: 参考 README.md 中的生产环境部署章节，推荐使用 Nginx + systemd。
